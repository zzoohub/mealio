# MVP Product Requirements Document

## Mealio - 사진 기반 식단 다이어리

**Version:** 1.1
**Date:** 2026-01-08
**Status:** MVP Development

---

## 1. 제품 개요

### 비전

"찍으면 끝" - 사진 한 장으로 완성되는 가장 간단한 식단 기록

### 핵심 가치

- **즉시성**: 사진 찍으면 시간/장소와 함께 바로 저장
- **부담 없음**: 메모, 입력 없이 사진만으로 기록 완료
- **나중에 추가**: 필요하면 언제든 정보 추가 가능

---

## 2. 문제 정의

### 사용자 문제

1. **기록의 번거로움**: 음식 검색, 양 입력 등 식사 기록에 시간 소요
2. **지속성 부족**: 복잡한 입력 과정으로 인해 기록을 포기
3. **즉시성 부재**: 먹기 전에 여러 정보를 입력해야 함

### 해결책

사진만 찍으면 끝. 시간/장소 자동 저장. 나머지는 나중에.

## 3. 화면 구성

### 3.1 메인 화면 (카메라)

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│            [카메라 뷰]               │
│                                     │
│                                     │
│                                     │
│            [📷 촬영]                 │
│                                     │
│  [다이어리]                     [앨범]   │
└─────────────────────────────────────┘
```

- **메인**: 카메라 뷰 (전체 화면)
- **중앙**: 촬영 버튼
- **좌측 하단**: 피드 버튼 (기록 보기)
- **우측 하단**: 앨범 버튼 (갤러리에서 선택)

### 3.2 다이어리 화면 (미정)

```
┌─────────────────────────────────────┐
│  < 뒤로                    [설정]   │
├─────────────────────────────────────┤
│  [일] [월] [화] [수] [목] [금] [토]  │  ← 주간 캘린더
│       ●        ●   ●              │  ← 기록 있는 날 표시
├─────────────────────────────────────┤
│                                     │
│  ┌─────┐  ┌─────┐  ┌─────┐        │
│  │ 사진 │  │ 사진 │  │ 사진 │        │  ← 해당 날짜 사진들
│  └─────┘  └─────┘  └─────┘        │
│  12:30     14:20     19:15         │
│  강남역     회사       집            │
│                                     │
└─────────────────────────────────────┘
```

- **상단**: 주간/월간 캘린더 (날짜 선택)
- **하단**: 선택된 날짜의 사진 피드
- 날짜 탭하면 해당 날짜 기록으로 필터링

### 3.3 핵심 플로우

**사진 기록**
```
카메라 → 촬영 → 즉시 저장 (시간/장소 자동)
                    ↓
         (백그라운드) AI 분석 → 영양 정보 추가
```

**앨범에서 추가**
```
앨범 버튼 → 갤러리 → 사진 선택 → 저장
```

**과거 기록 확인**
```
다이어리 버튼 → 캘린더에서 날짜 선택 → 사진 탭 → 상세 보기
```

## 4. 기능 요구사항

### 4.1 카메라 (메인)

| ID | 기능 | 설명 | 우선순위 |
|---|---|---|---|
| CAM-01 | 사진 촬영 | 앱 내 카메라로 음식 사진 촬영 | P0 |
| CAM-02 | 즉시 저장 | 촬영 즉시 저장 (추가 입력 없음) | P0 |
| CAM-03 | 시간 자동 기록 | 촬영 시간 자동 저장 | P0 |
| CAM-04 | 장소 자동 기록 | GPS 기반 위치 자동 저장 | P0 |
| CAM-05 | 앨범 선택 | 갤러리에서 사진 선택하여 추가 | P0 |
| CAM-06 | 다이어리 이동 | 다이어리 화면으로 이동 버튼 | P0 |
